Poppy

Free POP3 Client Library for .NET applications

Download this library as a .zip file Download this library as a tar.gz file

About

Poppy is a small free .NET library for interacting with POP3 mail servers.

Features

  • RFC 1939-compliant
  • Optional POP3 commands support
  • SSL support
  • APOP authentication support
  • Lightweight (~20KB)
  • Thread-safe
  • Custom POP3 transport support

Supported Commands

The library supports the following POP3 commands: STAT, LIST, UIDL, USER, PASS, APOP, RETR, TOP, DELE, NOOP, RSET, QUIT.
